I&IT Accessibility Centre of Excellence

WCAG 2.0 Level A & AA Scavenger Hunt


Learn more about the Scavenger Hunt

Purpose:

A learning experience to better understand how to test for accessibility barriers and compliance.

Who is this Scavenger Hunt for:

The scavenger hunt is for anyone looking to learn more about common web accessibility barriers. You do not need to have knowledge of HTML, CSS, JavaScript or ARIA.

System Requirements:

As part of the Hunt, there are some free open-source tools that must be downloaded to begin. To test, we require the download of one screen reader, which will be used to emulate the experience of a screen reader user. We recommend the download of NVDA screen reader as it is free. Note: Tools are free of charge, have been certified by the IT Service Desk and can be requested via SODO. We will also test using other common browser extensions and tools which are listed under the next page under "Your Toolbox".

Disclaimer: This Scavenger Hunt includes common web accessibility issues for the purpose of training only.
