Required Documents

The following documents are recommended when using the Web Accessibility Scavenger Hunt. They are recommendations and not requirements for participation.

Web Content Accessibility Guidelines (WCAG 2.0) Success Criteria Checklist

The WCAG Checklist that we recommend using is a simple 3 page checklist that shows many of the common issues in checklist format. Our suggested checklist is the WebAim WCAG Accessibility Checklist (PDF format)

ACOE Web Testing Reporting Document

Our simple WCAG Accessibility Compliance Report spreadsheet can be used to fill out specific information regarding the accessibility issues you encounter. For each Criteria that is listed as a fail in the spreadsheet, add details in the defect details column on the accessibility barriers you encounter. ACOE report template (Excel format)

Tips on Using JAWS and/or NVDA

Our one-page tip sheet on using the JAWS and NVDA screen reader to read through web-content JAWS and NVDA Tip Sheet (Word format)
